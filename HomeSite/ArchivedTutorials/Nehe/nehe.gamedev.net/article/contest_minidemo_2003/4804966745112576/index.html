Contest: Mini-Demo 2003

Welcome To The Sixth NeHe Productions OpenGL Challenge!

Theme: Anything You Want!!!

Rules:

  • All demos must use OpenGL.
  • Full source code must be included with each entry.
  • Code must compile as is. Code that will not compile will not win.
  • All files must be compressed with ZIP or RAR. Do not send Ace, Tar, Etc.
  • The source should be in it's own zip file and must include all files required to compile and run.
  • The exe should be in it's own zip file and must include all files required to run.
  • Freeware libraries are allowed, but remember to include any files required to compile the source.
  • The maximum size of your compiled demo and all required libraries and data files can be no more than 500k.
  • The demo must run without any user interaction from start to finish.
  • The demo should run for at least one minute without repeating. After a minute you can loop if you want.
  • Try to include as many effects as possible while staying under 500k.
  • Code should have some type of commenting (even if it's minimal).
  • If you use extensions, you better have something to fall back on if the card doesn't support it!
  • There is no target hardware, your demo should run bearably at the very least on a 450MHz system w/ TNT2.
  • This contest is open to everyone!

Summary:

Create an amazing, mindblowing, thrilling, exciting, super cool mini demo. It must be under 500k in size including graphics, sound and any libraries. It should run for at least 60 seconds without repeating. A tune of some sort will get you bonus points. The demo should run without keyboard interaction (aside from a key to quit and one to toggle fullscreen). There is no theme... be creative!!! Source code must be included with your demo, and it MUST compile without issues. The demos will be posted Friday, January 31st. A winner will be selected based on visitor feedback and a small judging panel. The prize is a BRAND NEW... STILL IN THE PACKAGE copy of MindCandy!!! One of the coolest and most hard to get ahold of DVD's out there! 230 minutes of Demos on a double sided DVD (oldschool and new), Interviews with big names in the scene, Video from Assembly, Audio commentary on all demos, A 12 page full color booklet, Animated chapter menus and Hidden demos / easter eggs. Thanks to |Laz| for the contest idea!
